# WalkMap

WalkMap is a web-based walking route generator and distance tracking application developed as part of CSCI 412: Senior Seminar.

## Tech Stack
- Frontend: React (JavaScript)
- Backend: ASP.NET Core (.NET, C#)
- Database: Azure SQL / Azure Cosmos DB
- Deployment: Microsoft Azure
- Version Control: Git & GitHub
